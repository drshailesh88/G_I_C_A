# Session 11: React Email — Email Template Patterns
**Module:** Email Communications (#6)
**URL:** https://react.email/templates
**Time:** 30 minutes

## Template Gallery Review
For EACH template category document:
- Layout structure
- Sections: header, greeting, body, CTA button, footer
- Branding: logo placement, color usage
- Typical email width
- Responsive design approach

## GEM India Use Case Matches
Find and screenshot templates matching:

1. **REGISTRATION CONFIRMATION** — "welcome" or "signup confirmation"
2. **ITINERARY/TRAVEL** — "booking confirmation" or "travel details"
3. **ROLE ASSIGNMENT** — "invitation" or "you've been assigned"
4. **CERTIFICATE DELIVERY** — "document attached" or "download your certificate"

For each match note:
- Exact layout structure (what comes first, second, third)
- Personalization variable usage
- Attachment/download link presentation
- Tone and length of copy
